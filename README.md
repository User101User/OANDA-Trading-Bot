# OANDA-Forex-Bot
A simple Python bot that fetches candlestick data from OANDA, calculates technical indicators, and can place market orders.



Features

    -Fetch candles: Loads OHLC data for any instrument and timeframe.
  
    -Indicators: Adds EMA 5, EMA 8, and ATR 14 using pandas_ta.
  
    -Order execution: Sends market orders with stop-loss and take-profit.

Requirements

    -oandapyV20, pandas, pandas_ta
  
    -OANDA API key + account ID
